Feasibility

对比 BLIP-base / BLIP-large / GIT / OFA 多个模型方案
选定 BLIP-base：约 1GB，CPU 推理 2-5 秒/图，平衡体积与速度

模块级设置 HF_ENDPOINT 国内镜像，规避 SSL 错误
三级模型路径查找：环境变量 → HF 缓存 → 网络下载
依赖注入传入 PDFAltTextGenerator，保持模块解耦
